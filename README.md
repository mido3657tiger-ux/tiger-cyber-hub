# 🐅 TIGER CYBER HUB

المستودع الرسمي والمنصة البرمجية الخاصة بـ **Tiger Cyber Hub** لعرض المشاريع المؤتمتة، حلول الويب الاحترافية، وأنظمة الأمن السيبراني.

## 🚀 نظرة عامة على المشاريع (Portfolio)
يحتوي هذا المستودع على البنية الأساسية للموقع التعريفي ومعرض الأعمال الذي يضم:
* **🤖 نظام TigerX_Empire الذكي:** بوتات تليجرام متطورة وأنظمة أتمتة وسيرفرات مبنية بـ Node.js.
* **🛒 متجر Selfie Store الإلكتروني:** تصميم ويب متكامل ومتجاوب بالكامل مع الموبايل لبيع المنتجات والهواتف.
* **🛡️ أدوات Tiger-CORE:** حزمة سكربتات مخصصة لفحص الشبكات، اختبار الاختراق، وتقييم الحماية للمواقع والسيرفرات.
* **⚡ أنظمة NovaDev:** سكربتات ربط الـ APIs وبوابات الدفع ومعالجة البيانات التلقائية.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **Frontend:** HTML5, CSS3 (Cyberpunk High-Contrast Theme), JavaScript (Vanilla)
* **Backend & Automation:** Node.js, Telegram Bot API
* **Environment & Testing:** Termux, Linux, Pentesting Tools (Nmap, SQLmap)
* **Hosting & Deployment:** GitHub, Cloudflare Pages
